# monthly_report

一人暮らしのメトリクスを格納するテーブル

| カラム名  | 型          | PRIMARY KEY | NOT NULL | 値の説明                                                     |
|--------|-------------|-------------|----------|-----------------------------------------------------------|
| date   | DATETIME    | ○           | ○        | 日時                                                        |
| type   | VARCHAR(30) | ○           | ○        | 出費種別．以下は例．<br/><ul><li>食費</li><li>電気代</li></ul> |
| amount | INT         |             | ○        | 金額                                                        |
